# Zenelait LMS – Complete Learning Management System Project

This repository contains the complete codebase for the Zenelait Learning Management System (LMS), consisting of a Spring Boot backend and a React-based frontend.

## Project Structure

- **`lms-backend/`**: Spring Boot backend application (Java 17, Spring Boot 3.2, Spring Security, JWT, JPA/Hibernate, MySQL/TiDB).
- **`lms-clean-components/`**: Clean UI components for the frontend application (React, Tailwind CSS, etc.).

## Getting Started

For detailed setup instructions, refer to the individual module directories:
- For Backend instructions, see `lms-backend/lms-backend/README.md`.
- For Frontend/Certificate features, see `README_CERTIFICATES.md`.
